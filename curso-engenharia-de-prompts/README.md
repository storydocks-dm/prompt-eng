# Curso Completo de Engenharia de Prompts com Claude

Bem-vindo ao Curso Completo de Engenharia de Prompts!

Este curso foi desenvolvido para fornecer um guia abrangente, desde os conceitos básicos até as técnicas avançadas de engenharia de prompts, utilizando como base o material da Anthropic. O objetivo é capacitar você a interagir de forma eficaz e eficiente com modelos de linguagem como o Claude.
